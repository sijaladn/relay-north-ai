# Relay North AI Landing Page

Static landing page for Relay North AI, a Toronto-focused AI training and workflow automation consultancy.

## Publish With GitHub Pages

Use the repository root as the GitHub Pages source:

- Source: Deploy from branch
- Branch: `main`
- Folder: `/root`

The booking CTA points to https://calendly.com/relaynorth-ai and the email form opens a draft to relaynorth.ai@gmail.com.
